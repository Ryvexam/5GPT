Guide de lancement du projet 5GPT
Application disponible également sur : https://ai.ryvexam.fr/
Installation

    git clone https://github.com/Ryvexam/5GPT.git
    cd 5GPT
    npm i
    npm run dev

L’application sera ensuite accessible localement :
http://localhost:3000

------------------------------------------------------------------------

Clé API requise

Au premier lancement, l’application demandera une clé d’API compatible
Mistral ou OpenAI.

Pour effectuer des tests, une clé Mistral est disponible :

    RTy4yfRRNG5EkLqsfaspYyJAe4Iz5pKH

------------------------------------------------------------------------

Documentation intégrée

Rendez-vous sur l’URL :
http://localhost:3000/docs

En bas de la page se trouvent trois boutons permettant d’accéder à :

-   Les tests effectués (ex: prompt injection, etc.)
-   Les résultats de ces tests
-   Le guide d’implémentation des bonnes pratiques de sécurité

------------------------------------------------------------------------

Versions des prompts

Les différentes versions des prompts sont consultables à l’adresse :
http://localhost:3000/prompts

Pour chaque outil, les versions suivantes sont disponibles :

-   v0
-   v1
-   vFinal

Chaque outil dispose de ces trois versions.
